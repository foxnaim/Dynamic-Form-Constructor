## About

A powerful form builder that lets you create dynamic forms with drag-and-drop functionality. Add fields, configure validation rules, and generate forms on the fly.

## Features

- 🖱️ **Drag & Drop** — Intuitive form field placement
- 📝 **Field Types** — Text, select, checkbox, radio, date, and more
- ✅ **Validation** — Configure rules per field
- 📋 **Preview** — Live form preview
- 💾 **Export** — Save form configurations
- 🎨 **Customizable** — Style forms to match your brand

## Supported Field Types

| Type | Description |
|------|-------------|
| Text | Single-line text input |
| Textarea | Multi-line text input |
| Select | Dropdown selection |
| Checkbox | Multiple choice |
| Radio | Single choice |
| Date | Date picker |
| Number | Numeric input |

## Getting Started

```bash
git clone https://github.com/foxnaim/Dynamic-Form-Constructor.git
cd Dynamic-Form-Constructor
yarn install
yarn dev
```
